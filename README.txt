A ruby script to generate a Obstacle Course for Bzflag http://www.bzflag.org

GitHub https://github.com/BZFlag-Dev/bzflag

Fill the obs folder with obstacles more info at:

https://forums.bzflag.org/viewtopic.php?f=66&t=20160&p=176707#p176707

Run ./obstacle-keep.sh

It will generated the the course and place obstacles randomly in placeholders.

To change the number of obstacles generated change:

mxcnt=25
